
# Advanced Database Applications

## Employee Wages Tracking Application

This project was developed as the final project for the **Advanced Database Applications** course.

The project focuses on designing and implementing a relational database for employee and wage management using Microsoft SQL Server. It includes database modeling, SQL scripting, data manipulation, and reporting queries.

## Technologies Used

- Microsoft SQL Server 2022
- T-SQL
- Relational Database Design
- Entity Relationship (ER) Modeling

## Features

- Employee Management
- Department Management
- Wage Tracking
- User Management
- SQL Queries and Reporting
- Foreign Key Relationships
- Data Manipulation (INSERT, UPDATE, DELETE)

## Database Structure

- Employees
- Departments
- Employee Wages
- Users
- Lookups

## Learning Outcomes

- Relational Database Design
- SQL Query Development
- Database Normalization
- Data Analysis
- Problem Solving
